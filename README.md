# MoodSpace

MoodSpace is a simple React app that suggests content based on the userâ€™s mood.

The user selects a mood (happy, sad, relaxed, angry), and the app shows
recommendations such as music and movies related to that mood.

## Features

- Mood selection with buttons
- Movie recommendations using a public API (TMDB)
- Simple and clean user interface
- Blurred background for better visual experience

## Built With

- React
- Vite
- CSS
- TMDB API

## How to Run the Project

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the development server:
   npm run dev

## Notes

This project was built as a learning project to practice React, components,
state management, and working with public APIs.
